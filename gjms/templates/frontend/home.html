This is an example frontend page we setup for you. You can keep it this way, or build your own. Whatever fits your needs!

Be creative!

We have provided a few macros, modules and CSS classes for you to try. These try to make setting up a cool game jam page as painless as possible.

But if you're not happy with them, you can just look through the templates folder and edit them to your liking.

Of course you can also write a new page form scratch. We don't mind.

Behind the scenes!

Another big feature of GJMS is the backend. You can access it here.

The standard username and password are root. You'll be asked to change the password after the first login.

You can then create a new administrative user in the backend, though you can not delete the root user for security purposes.
